New extension NEW TAB MANAGER to manage "New Tab" extensions.

  • This extension is an evolution of the "Origina New Tab" extension you are using.
  • You will be able to assign to the extension icon the extension of new tab you want, not only the original chrome, so you can easily use two "new tab" extensions at the same time.
  • From the options page of the extension you can install/uninstall extensions and test new pages momentarily without assign to any icon.
